# Project 1 - Application portal for a dental clinic

I designed website for the dentists so that they can send requests for what kind of dental prosthesis they want from the dental clinic I worked for. I used Bootstrap for the frontend and Laravel for the backend. Website interface changes based on the login information. If administrators login, they can create/update/delete dentists, accept dentist applications or change product prices. To see the images of the project you can click on the "Project 1” folder.

# Project 2 - Social media app of an organisation

I designed mobile app and admin panel for an organisation. I used Laravel for the backend and designed an admin panel using Bootstrap within its blade files. The admin panel enables administrators to create/delete/update users, organization events or announcements and send notifications to devices with the mobile app installed. I utilized Flutter for the mobile app that lets users login/register, see the events/announcements, find other users. Users can also find other users so that they can follow and message to each other. To see the images of the project you can click on the "Project 2” folder.
